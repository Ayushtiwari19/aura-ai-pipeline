# AuraGen

AuraGen is a modern AI assistant platform built with **Next.js**, **React**, **TypeScript**, and **Tailwind CSS**. It brings together an intelligent chatbot, an interactive dashboard, a code editor, and real-time analytics in a single frontend application.

## Features

- 🤖 **Intelligent Chatbot** — conversational AI interface for user interaction
- 📊 **Interactive Dashboard** — real-time analytics and data visualization
- 💻 **Code Editor** — in-browser code editing experience
- ⚡ **Modern Stack** — built on Next.js App Router with TypeScript and Tailwind CSS for fast, type-safe, responsive UI

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js |
| UI Library | React |
| Language | TypeScript |
| Styling | Tailwind CSS |

## Getting Started

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

## Development Approach

This project was built using a combination of hands-on development and AI-assisted coding tools (e.g., Claude, ChatGPT-style assistants). AI tools were used to accelerate scaffolding, boilerplate generation, and iteration speed — while architecture decisions, feature design, debugging, and final review were done manually.

This is a common and increasingly standard workflow in modern software development. Using AI tools effectively — knowing what to ask for, reviewing the output critically, and understanding every part of the resulting codebase — is itself a skill, and one I'm comfortable discussing and demonstrating in detail.
